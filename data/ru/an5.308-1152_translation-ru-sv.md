*Ангуттара Никая 5*

*29\. Страсть и далее — серия повторений*

Ради полного понимания страсти … ради полного уничтожения страсти … ради оставления страсти … ради уничтожения страсти … ради преодоления страсти … ради угасания страсти … ради прекращения страсти … ради отказа от страсти … ради покидания страсти следует развивать пять вещей\.

Ради полного понимания … полного уничтожения … оставления … уничтожения … преодоления … угасания … прекращения … отказа … покидания злобы … заблуждения … злости … враждебности … клеветы … презрения … зависти … скупости … неискренности … лукавства … упрямства … горячности … самомнения … надменности … опьянения … беспечности следует развивать пять вещей\.

Какие пять? Силу веры, силу усердия, силу осознанности, силу сосредоточения, силу мудрости\. Ради покидания беспечности следует развивать эти пять вещей»\.
