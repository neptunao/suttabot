*Восходящие Наставления 6\.111*

*Глава Тика*

*Сутта Элементы*

“Монахи, есть эти три вещи\. Какие три? Элемент чувственности, элемент недоброжелательности, элемент причинения вреда\. Таковы три вещи\. Три иные вещи следует развить ради отбрасывания этих трёх вещей\. Какие три? Элемент отречения следует развить для отбрасывания элемента чувственности\. Элемент доброжелательности следует развить для отбрасывания элемента недоброжелательности\. Элемент безвредности следует развить для отбрасывания элемента причинения вреда\. Эти три вещи следует развить ради отбрасывания первых трёх вещей”\.

Пятая\.
